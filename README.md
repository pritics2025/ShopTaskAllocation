# Shop Task Allocation System

## Overview

A web-based application developed to manage and assign shop tasks efficiently. The system helps organize work, track task assignments, and improve workflow.

## Features

- User Login
- Task Assignment
- Task Management
- Employee Dashboard
- Database Integration

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Bootstrap
- Eclipse IDE

## Architecture

MVC2 Architecture
